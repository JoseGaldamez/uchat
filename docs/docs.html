Documentación

Las aplicaciones de chat son un buen ejercicio para aprender a utilizar una tecnología. Ofrecen muchos desafíos que pueden hacer que tendendamos mejor el flujo de trabajo de los datos. En este caso un equipo de 3 estudiantes de la Universidad Tecnológica de Honduras hemos creado una pequeña aplicación en Java que conecta con un backend en Firebase.

En la página principal de este sitio web puede descargar el proyecto completo de Java Maven y ejecutar la aplicación corriendo el archivo llamado "run". Esto ejecuta un script que hace el llamado al archivo .jar donde se encuentra la aplicación compilada.

Si desea verificar el código puede utilizar un software como Apache NetBeans para abril el proyecto. Después de abrirlo podrá compilarlo y modificarlo a su gusto.

Un archivo importante es el que se encuentra en la raiz del directorio del proyecto, me refiero a "uth-chat-firebase.json" que contiene la información para conectar con la base de datos. Lo ideal es que reemplaces este archivo por el que te proporcione tu proyecto de Firebase. Reemplazalo con el mismo nombre y cuando corras la aplicación creará la conexión con tu base de datos y no con la que viene por defecto a la cual no tienes acceso.

Para terminar queremos agracer al ingeniero Hector Conrado Lanza A. de la UTH, un catedrático capaz que nos motiva y nos enseña a ser mejores programadore en casa clase que imparte.

Cualquier duda o sugerencia puedes hacer las en la sección de Issues del repositorio de Github y con gusto estaremos corrigiendo o integrando modificaciones. Gracias.
